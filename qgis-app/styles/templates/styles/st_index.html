QGIS Style Repository

QGIS Style Repository is a place where styles of the layers in QGIS can be shared. The styles canbe Exported through the "Save Style" button in the Properties dialog, Style tab and can be Imported using the "Load Style" button.

To upload a new style go to Style Upload Form and upload the .qml file exported by the "Save Style" option.
